# Sistema Comercial em Clipper

Este repositório contém o código-fonte de um sistema comercial desenvolvido originalmente em **1995**, utilizando a linguagem **Clipper**.

O sistema foi utilizado em ambiente real por clientes até **2022**, contemplando módulos como:

* Estoque
* Compras
* Vendas
* Assistência técnica
* Contas a pagar
* Contas a receber
* Extratos bancários

A publicação deste repositório tem finalidade **histórica, educacional e de preservação da minha trajetória profissional no desenvolvimento de software**.

Este projeto representa uma parte importante do início da minha carreira na área de tecnologia e documenta uma época em que sistemas comerciais desktop, baseados em arquivos locais e interfaces texto, eram amplamente utilizados em empresas de pequeno e médio porte.

## Aviso importante

O código é disponibilizado **sem garantia de funcionamento, suporte, manutenção ou adequação para uso em produção**.

Por se tratar de um sistema legado, desenvolvido em tecnologias antigas, podem ser necessários ajustes no ambiente, dependências ou configuração para compilação e execução.

## Requisitos para compilação

Para compilar o projeto, é necessário ter instalado:

* Borland C++ Compiler 5.5 — BCC55
* xHarbour

## Building

```bash
hbmake comercio.bc
```

## Running

```bash
comercio.exe
```
