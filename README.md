Este projeto contém o código-fonte de um sistema comercial desenvolvido originalmente em 1995 em Clipper.

O sistema foi utilizado em ambiente real por clientes até 2022, contemplando módulos de estoque, compras, vendas, assistência técnica, contas a pagar, contas a receber e extratos bancários.
A publicação deste repositório tem finalidade histórica, educacional e de preservação da minha trajetória profissional no desenvolvimento de software.
O código é disponibilizado sem garantia de funcionamento, suporte, manutenção ou adequação para uso em produção.
